# 采购系统-食材端 微信小程序

基于微信小程序开发的食材采购管理客户端，支持食材清单管理、采购任务跟踪、历史采购记录和客户名单管理。

## 功能模块

### 🍅 食材清单
- 添加、编辑、删除食材
- 支持分类筛选（冻品、酱料、蔬菜、肉类）
- 搜索功能
- 数据导出

### 🛒 采购任务
- 从食材清单选择采购项
- 支持自定义数量和单价
- 状态管理（未买、缺货、已买）
- 一键完成采购

### 📜 历史采购
- 采购记录查询
- 时间筛选（当天、近3天、近7天、近15天、本月）
- 采购统计

### 👥 客户名单
- 客户信息管理
- 支持搜索
- 包含联系方式、地址等信息

## 技术栈

- 微信小程序
- JavaScript
- LocalStorage 数据存储

## 项目结构

```
miniprogram/
├── app.js              # 应用入口
├── app.json            # 应用配置
├── app.wxss            # 全局样式
├── sitemap.json        # sitemap配置
├── project.config.json # 项目配置
├── pages/
│   ├── ingredients/    # 食材清单页面
│   ├── purchase/       # 采购任务页面
│   ├── history/        # 历史采购页面
│   └── customers/      # 客户名单页面
└── images/             # 图片资源
```

## 使用说明

1. 克隆项目到本地
2. 使用微信开发者工具打开 `miniprogram` 目录
3. 在微信开发者工具中进行调试和预览

## 注意事项

- 需要在微信公众平台注册小程序账号
- AppID 需要在 `project.config.json` 中配置
- 图标文件需要自行添加到 `images` 目录
